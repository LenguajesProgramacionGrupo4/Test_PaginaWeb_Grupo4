# MIDI 4 ALL
## Plataforma web para compartir archivos midi
si eres musico, ingeniero de sonido o te mueves en el mundo de la produccion musical este lugar es para ti.
MIDI 4 ALL es un espacio digital donde los usuarios pueden compartir reproducir y descargar archivos de formato midi

### Descripción 

Este proyecto es una pagina web hecha para subir nuestra propuesta de proyecto MIDI 4 All, utilizando herramientas como HTML para estructurar elcontenido de la página web y CSS para el diseño y la presentación de la pagina.
